# Platform Compliance Rules

> **Priority:** HIGH
> **Applies to:** SEO Writer, all production agents
> **Enforcement:** Pre-publish verification required

---

## YouTube Compliance

### Community Guidelines

```yaml
required:
  - No harmful or dangerous content
  - No hateful content
  - No harassment or cyberbullying
  - No violent or graphic content
  - No misleading content
  - No spam or deceptive practices

content_ratings:
  - Accurate age designation
  - Appropriate content warnings
  - Honest titles and thumbnails
```

### YouTube Kids Requirements

```yaml
made_for_kids:
  must:
    - Mark as "Made for Kids" if targeting children
    - No personalized ads
    - No comments enabled
    - No live chat
    - No notification bell
    - No community tab

  content_requirements:
    - Educational or entertaining for children
    - No mature themes
    - No violence (even cartoon)
    - No fear-inducing content
    - No commercial content
    - No inappropriate animations
```

### Metadata Requirements

```yaml
titles:
  - Accurate and descriptive
  - No clickbait
  - No misleading claims
  - Include relevant keywords
  - Under 100 characters

descriptions:
  - Accurate content summary
  - Age-appropriate language
  - Include relevant keywords
  - Include proper credits
  - No excessive hashtags

thumbnails:
  - Accurate representation
  - No misleading imagery
  - Age-appropriate
  - Clear and readable
  - No text that contradicts content

tags:
  - Relevant to content
  - No misleading tags
  - Include Vietnamese and English
  - No competitor brand names
```

## TikTok Compliance

### Community Guidelines

```yaml
required:
  - No violent or graphic content
  - No dangerous activities
  - No hate speech
  - No harassment
  - No adult content
  - No illegal activities
  - No spam or fake engagement

minor_safety:
  - Extra protection for content with minors
  - No duets/stitches with minor content
  - Age-appropriate interactions
```

### Content Requirements

```yaml
format:
  - Vertical video (9:16)
  - Clear audio
  - No watermarks from other platforms
  - Original or properly licensed content

metadata:
  - Accurate descriptions
  - Relevant hashtags (not excessive)
  - Proper sound attribution
  - Age-appropriate caption
```

## Copyright Compliance

### Music

```yaml
requirements:
  - Use royalty-free music only
  - Verify license before use
  - Credit properly in description
  - Check for regional restrictions

sources:
  - YouTube Audio Library (free)
  - Licensed stock music
  - Original compositions
  - Public domain music
```

### Images/Video

```yaml
requirements:
  - Original AI-generated content
  - Public domain sources verified
  - Proper attribution
  - No copyrighted characters

verification:
  - Legal Checker agent reviews all sources
  - Document licenses in project files
```

### Stories/Scripts

```yaml
requirements:
  - Public domain stories only
  - Verify copyright expiration
  - Document source and date
  - Note any adaptations made

examples_allowed:
  - Classic fairy tales
  - Traditional folk stories
  - Ancient myths/legends
  - Out-of-copyright literature
```

## Pre-Publish Checklist

### YouTube

- [ ] Content appropriate for designated audience
- [ ] "Made for Kids" properly set
- [ ] Title accurate and non-clickbait
- [ ] Description complete and accurate
- [ ] Thumbnail appropriate and accurate
- [ ] Tags relevant and complete
- [ ] Music/audio properly licensed
- [ ] No copyright claims anticipated
- [ ] Category correctly selected
- [ ] Language correctly set

### TikTok

- [ ] Vertical format (9:16)
- [ ] Under platform time limits
- [ ] Audio properly attributed
- [ ] No prohibited content
- [ ] Hashtags relevant
- [ ] Description appropriate
- [ ] No other platform watermarks

### General

- [ ] Content reviewed by QA Reviewer
- [ ] Legal Checker sign-off
- [ ] All licenses documented
- [ ] Credits included
- [ ] Safety guidelines met

## Platform-Specific Best Practices

### YouTube

```yaml
optimization:
  - First 48 hours critical for algorithm
  - Consistent posting schedule
  - End screens and cards
  - Playlist organization
  - Community engagement (if not Kids)

avoid:
  - Clickbait that doesn't deliver
  - Misleading thumbnails
  - Engagement bait
  - Sub-for-sub mentions
```

### TikTok

```yaml
optimization:
  - Hook viewers in first second
  - Trending sounds when appropriate
  - Consistent posting time
  - Engaging captions
  - Reply to comments

avoid:
  - Banned hashtags
  - Excessive hashtags
  - Low-quality video
  - Watermarks from other apps
```

## Violation Response

### Level 1: Warning

```
Action: Review and fix content
Timeline: Before next publish
Documentation: Note in project log
```

### Level 2: Revision Required

```
Action: Remove or edit violating content
Timeline: Immediate
Documentation: Full incident report
```

### Level 3: Account Risk

```
Action: Stop all publishing, full audit
Timeline: Until cleared
Documentation: Comprehensive review of all content
```
