# PiggyTales Commands Reference

> **Version:** 1.0.0
> **Alias:** `/t` = `/tale`

---

## Quick Reference

| Category | Commands |
|----------|----------|
| **Project** | `new`, `status`, `list`, `open`, `archive`, `clone`, `delete`, `info`, `history`, `timeline` |
| **Phase** | `phase`, `next`, `back`, `jump`, `skip`, `unskip`, `redo`, `phases`, `progress` |
| **Workflow** | `workflow`, `complexity`, `preset` |
| **Review** | `council`, `score`, `report` |
| **Agent** | `agent`, `agents` |
| **Audience** | `audience` |
| **Content** | `source`, `script`, `scenes`, `chapters` |
| **Production** | `mode`, `voice`, `image`, `video`, `sfx`, `audio` |
| **Export** | `export` |
| **Config** | `config` |
| **Utility** | `help`, `debug`, `reset`, `save`, `load`, `quick`, `continue`, `pause`, `resume` |
| **Batch** | `batch`, `series` |

---

## Shortcuts

```bash
/t s          # /tale status
/t n          # /tale next
/t a          # /tale approve
/t c          # /tale council
/t r          # /tale report
/t f "text"   # /tale feedback "text"
```
