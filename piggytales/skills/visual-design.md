# Skill: Visual Design

> **Used by:** Art Director, Prompt Engineer, Thumbnail Designer

---

## Overview

Visual Design skill enables creation of compelling visual content direction and AI prompts.

---

## Capabilities

- Style guide creation
- Character design
- Color theory application
- Composition planning
- Visual consistency
- Thumbnail optimization

- Create visual direction documents
- Develop consistent style guides
- Design compelling thumbnails
- Ensure age-appropriate visuals
