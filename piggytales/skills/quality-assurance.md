# Skill: Quality Assurance

> **Used by:** QA Reviewer, Council agents, all agents (verification)

---

## Overview

Quality Assurance skill enables systematic verification of content quality, safety, and compliance.

---

## Capabilities

- Content safety verification
- Quality checklist execution
- Platform compliance checking
- Technical validation
- Final approval process

---

## Checklist Types

```yaml
content_safety:
  - No prohibited content
  - Age-appropriate language
  - Positive messaging
  - No harmful themes

platform_compliance:
  - YouTube guidelines
  - TikTok guidelines
  - "Made for Kids" rules
  - Metadata accuracy

technical_quality:
  - File specifications
  - Naming conventions
  - Output completeness
  - Format validation

creative_quality:
  - Story coherence
  - Visual consistency
  - Audio quality
  - Engagement potential
```

---

## Quality Metrics

```yaml
scoring:
  viral_score:
    metrics: [hook, emotion, share, trend, unique]
    threshold: 6.5 minimum

  quality_gate:
    pass: "Meets all requirements"
    conditional: "Minor issues"
    fail: "Major issues"
```

---

## Process

1. Run automated checks
2. Perform manual review
3. Document findings
4. Make decision
5. Issue sign-off or return
