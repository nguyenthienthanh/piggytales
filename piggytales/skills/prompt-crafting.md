# Skill: Prompt Crafting

> **Used by:** Prompt Engineer, SFX Producer, Image/Video Producers

---

## Overview

Prompt Crafting skill enables creation of optimized prompts for AI generation tools.

---

## Capabilities

- Image prompt engineering
- Video prompt creation
- SFX prompt writing
- Platform optimization
- Consistency maintenance

---

## Prompt Types

```yaml
image_prompts:
  structure:
    - Subject description
    - Action/pose
    - Setting
    - Style
    - Technical specs
  platforms: [Midjourney, DALL-E, Stable Diffusion]

video_prompts:
  structure:
    - Opening frame
    - Motion description
    - Camera movement
    - Duration
  platforms: [Veo 3, Runway, Pika]

audio_prompts:
  structure:
    - Sound description
    - Style
    - Duration
    - Character
  platforms: [ElevenLabs SFX]
```

---

## Best Practices

- Be specific and detailed
- Maintain consistency keywords
- Include technical parameters
- Test and iterate
- Document successful patterns
